Rahmentyp

Wählen Sie den Rahmentyp, den Sie für Ihr Zertifikat verwenden wollen. Sie könenn auch einstellen, dass kein Rahmen gedruckt wird. Derzeitige Optionen sind:
Linien: Druckt eine Rahmen mit drei Linien rundum.
Fancy1 oder Fancy2: Druckt eine Schmuckrahmen.

Rahmenfarbe

Wenn ein Rahmentyp gewählt wurde, kann im nächsten Feld eine Farbe gewählt werden.
