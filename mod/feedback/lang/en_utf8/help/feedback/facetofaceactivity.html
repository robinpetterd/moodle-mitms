By selecting a face-to-face activity available in a course, the Excel export of the feedback analysis will include face-to-face session data as follows:

'summary' worksheet
Feedback will be grouped and analysed by each session that exists in the face-to-face activity.

'detailed' worksheet
Face-to-face session details (venue name, session start date, and session start time) will be included with each user's responses and relevant user profile data.
