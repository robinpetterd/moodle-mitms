Custom field filter

This filter allows you to filter competencies based on values of custom fields. The filter can be applied on a single custom field or an all custom fields. The filter has the following options:

  • contains - this option allows only competencies for which the specified field contains the text entered (if no text is entered, then the filter is disabled)
  • doesn't contain - this option allows only competencies for which the specified field does not contain the text entered (if no text is entered, then the filter is disabled)
  • is equal to - this option allows only competencies for which the specified field is equal to the text entered (if no text is entered, then the filter is disabled)
  • starts with - this option allows only competencies for which the specified field starts with the text entered (if no text is entered, then the filter is disabled)
  • ends with - this option allows only competencies for which the specified field ends with the text entered (if no text is entered, then the filter is disabled)
  • is empty - this option allows only competencies for which the specified field is equal to an empty string, but it is defined (the text entered is ignored)
  • is not defined - this option allows only competencies for which the specified field is not defined (the text entered is ignored)
  • is defined - this option allows only competencies for which the specified field is defined (the text entered is ignored)
